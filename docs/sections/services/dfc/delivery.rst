Delivery
========

Docker Container
----------------

**datafile** is delivered as a docker container that can be downloaded from onap:

    ``docker run -d -p 8100:8100 -p 8433:8433  nexus3.onap.org:10001/onap/org.onap.dcaegen2.collectors.datafile``
