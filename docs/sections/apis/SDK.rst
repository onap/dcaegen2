DCAE SDK
========

.. contents::
    :depth: 3
..

Overview
--------

DCAE SDK contains utilities and clients which may be used for fetching
configuration from CBS, consuming messages from DMaaP, etc. SDK is written in Java.

Artifacts
---------

Current version
~~~~~~~~~~~~~~~
.. code-block:: xml

    <properties>
        <sdk.version>1.4.1</sdk.version>
    </properties>
